# Glosario

### Expressions:
Código que al ser ejecutado resulta en un valor.

```jsx
'hello' + 'world'; // produce 'hello world'
a===20 && b===30; // produce true o false basado en los valores de a y b
```
---
### Variables:
Nombre que se le da a un espacio en memoria y que contiene un valor.

```jsx
let x; // se declara la variable de nombre x
x = 3; // se le asigna un valor de 3 a la variable de nombre x
let b = 5; // se declara una variable de nombre b y ademas se le asigna el valor numérico 5
```
