## Otra solucion al problema, con otra estructura repetitiva

<Steps>

<Step>

Hasta ahora resolvimos el contador con una estructura **for**, veamos otra solucion

</Step>
<Step>

Otra vez, a nuestro pseudocodigo le vamos a agregar una nueva funcionalidad,
la estructura repetitiva **mientras**

</Step>
<Step>

    x = 10;
    contador = 0;

    mientras (contador < x) {    // mientras 'contador' sea menor que 10
    
        contador = contador + 1

    }

</Step>
<Step>

Esta estructura 'mientras' a diferencia del 'repetir' repetira el bloque de codigo siempre
que la condicion sea verdadera y no necesita una variable auxiliar.
En este caso la condicion es (contador < x)

</Step>

</Steps>
